## Run locally
Open `index.html` in your browser.

## GitHub Pages
1. Create or use an existing repository.
2. Upload all files.
3. Go to `Settings` then `Pages`.
4. Choose `Deploy from a branch`.
5. Select the `main` branch and `/root`.
6. Save.

The `404.html` file is included for GitHub Pages behavior, and `.nojekyll` prevents Jekyll processing.
